Web前端程式設計師(FrontEnd) - 長期兼職/實習/可遠端 06/20更新

聯成數網股份有限公司 本公司其他工作
應徵
11~30人應徵

工作內容

聯成數網成立於2001年，我們是一群充滿創造力與熱情的研發團隊，我們十分熱衷軟體技術的精進與研發，並成功發展遊戲電商、印刷電商和知識服務等垂直領域電商平台 身為聯成的前端工程師，你將會接觸Web與APP跨平台裝置，也會與團隊成員一起參與前瞻技術平台的開發工作，只要你具備下列條件： 【工作內容】 - 開發各種前端程式 (Web / Mobile Web / APP) - UI畫面切版與前端程式套用 - 軟體產品開發與測試 - 團隊主管交付任務 【必要條件】 - 邏輯清楚，對軟體開發有高度熱情 - 具備資料結構與演算法基礎概念 - 具責任感與獨立思考能力，團隊配合度高 - 熟悉HTML + CSS + JS(ES6+)、RWD切版 - 對主流的前端框架 (React.js 或 Vue.js)有高度熱情 【加分條件】 - 已熟悉Node.js Web 開發框架(如Express / Koa) 【我們的產品】 聯成數網擅用RD長期研發的各種軟體技術，開發並上線各種新型網路服務： ■ Cupoy AI學習社群 www.cupoy.com ■ JColor 線上印刷電商(捷可印) www.jcolor.com.tw ■ H2PES 校園影音串流服務 https://www.h2pes.com/ ■ 9199 遊戲電商服務 www.9199.com.tw 【為什麼要選擇本工作】 - 你可以快速精進Web軟體技術，包括Web / Mobile App 到大型分散式運算環境。 - 你會跟非常優秀的技術咖一起共事。 - 如果你對軟體開發有高度熱情，聯成數網會是你很棒的選擇! 【本工作特色】 - 工作時間與地點自由安排，可來公司工作也可遠端完成工作任務，適合兼職與實習者 - 公司位於台北商業大學(善導寺站)，交通方便。 - 自由安靜的工作環境及工作氛圍。 【其它】 - 本工作屬於案件承攬性質，你可以自由選擇工作時間與工作地點。
